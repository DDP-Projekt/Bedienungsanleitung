# Duden/Ausgabe Funktionen
## Programm_Beenden
* Parameter: Code
* Parameter Typ(en): Zahl
* Rückgabe Typ: nichts

## Programm_Beenden_Standard
* Parameter: keine
* Parameter Typ(en): keine
* Rückgabe Typ: nichts

## Laufzeitfehler
* Parameter: Nachricht, Code
* Parameter Typ(en): Text, Zahl
* Rückgabe Typ: nichts

## Betriebssystem
* Parameter: keine
* Parameter Typ(en): keine
* Rückgabe Typ: einen Text

## Ist_Befehlszeile
* Parameter: keine
* Parameter Typ(en): keine
* Rückgabe Typ: einen Boolean
